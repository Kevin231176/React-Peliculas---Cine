Catálogo de Películas (React + Vite)

Aplicación web desarrollada con React y Vite que consume la API de The Movie Database (TMDB) para mostrar películas populares en un diseño moderno inspirado en Netflix.

Demo

Repositorio del proyecto:
https://github.com/Kevin231176/React-Peliculas---Cine.git

Tecnologías utilizadas
React
Vite
CSS (Grid y efectos hover)
API de TMDB
Características
Listado de películas populares
Imágenes dinámicas desde la API
Diseño inspirado en Netflix
Efectos hover con animaciones
Diseño responsive
Instalación

Clona el repositorio:

git clone https://github.com/Kevin231176/React-Peliculas---Cine.git

Entra al proyecto:

cd React-Peliculas---Cine

Instala dependencias:

npm install

Ejecuta el proyecto:

npm run dev

Configuración de API

Crea un archivo .env en la raíz del proyecto:

VITE_API_KEY=TU_API_KEY

Puedes obtener tu API en:
https://www.themoviedb.org/
