Proyecto Cine React

Aplicación web desarrollada con React que simula una plataforma de cine, donde los usuarios pueden visualizar películas, seleccionar asientos y realizar una compra simulada de entradas.

Demo en vivo

https://proyecto-cine-react.netlify.app

Funcionalidades
Visualización de películas populares desde API (TMDB)
Navegación entre vistas con React Router
Vista detallada de cada película
Selección interactiva de asientos
Contador dinámico de entradas
Cálculo automático del total a pagar
Interfaz de compra tipo cine
Sección de combos de comida
Diseño responsive para dispositivos móviles
Tecnologías utilizadas
React
JavaScript (ES6+)
CSS3
React Router DOM
API de The Movie Database (TMDB)
Características destacadas
Sistema de selección de asientos con estados:
Disponible
Seleccionado
Ocupado
Simulación de sala de cine con pantalla y distribución de asientos
Interfaz dividida en:
Información de la película
Zona de asientos
Panel de compra
Cálculo de precios en tiempo real
Instalación y uso

Clonar el repositorio:

git clone https://github.com/Kevin231176/React-Peliculas---Cine.git

Entrar al proyecto:

cd React-Peliculas---Cine

Instalar dependencias:

npm install

Ejecutar en modo desarrollo:

npm run dev

API

Este proyecto utiliza la API de TMDB para obtener las películas.

Puedes usar tu propia API key para ampliar o modificar funcionalidades.
